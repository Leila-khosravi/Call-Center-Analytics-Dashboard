# Call Center Performance Dashboard (Power BI)

## Overview

This Power BI dashboard provides operational insights into call center performance through interactive visualizations and KPI monitoring.

## Features

- Weekly call volume KPIs
- Hourly call volume analysis
- Peak-hour identification
- Trend analysis
- Interactive filtering
- Executive performance monitoring

## Tools Used

- Power BI
- Power Query
- DAX
- Excel
